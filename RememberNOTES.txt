Spring Boot mein kya hota tha? Tumne application.properties mein spring.datasource.url likha, aur Spring Boot ne khud usko padh liya aur connection bana diya.

** Python mein ye kaam 3 steps mein hota hai (jiske liye 3 files hain):

** .env (The Godown): Ye sirf ek simple text file hai jahan passwords aur URL safe rakhe jate hain (jaise application.properties).

** setting.py (The Delivery Boy/Security Guard): Python directly .env file ko nahi       samajhta. setting.py ka kaam hai .env file se data (URL) uthana, check karna ki wo sahi hai ya nahi, aur usko Python Variables mein convert karna.

** db.py (The Connection Engine): Ab setting.py ke paas URL aa gaya hai. db.py us URL ka use karke actual database (PostgreSQL) ke sath connection establish karta hai.



# pip install SQLAlchemy:---> this command is used to install the SQLAlchemy library, which is an Object-Relational Mapping (ORM) tool for Python. It allows developers to interact with databases using Python objects and classes instead of writing raw SQL queries. With SQLAlchemy, you can define your database schema using Python classes and then use those classes to perform CRUD (Create, Read, Update, Delete) operations on the database. This makes it easier to work with databases in Python and helps to improve code readability and maintainability.


# pip install pydantic_settings ---> this command is used to install the pydantic_settings library, which is a package that provides a way to manage application settings using Pydantic models. Pydantic is a data validation and settings management library for Python that allows you to define your application settings as Python classes with type annotations. With pydantic_settings, you can easily load settings from environment variables, configuration files, or other sources, and validate them using Pydantic's powerful validation features. This helps to ensure that your application settings are correct and consistent, and makes it easier to manage configuration in your application.


# fastapi dev main.py :-- command help krta hai ki hamare main.py file me koi changes hone par server automatically restart ho jayega aur hamare changes reflect ho jayenge. Isse hamara development process bahut hi smooth ho jata hai kyunki hame baar baar server ko manually restart karne ki jarurat nahi hoti hai. Is command ka use karne ke liye hame fastapi library ko install karna hoga, jiske baad ham is command ka use karke apne application ko run kar sakte hai.
