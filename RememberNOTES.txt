Spring Boot mein kya hota tha? Tumne application.properties mein spring.datasource.url likha, aur Spring Boot ne khud usko padh liya aur connection bana diya.

** Python mein ye kaam 3 steps mein hota hai (jiske liye 3 files hain):

** .env (The Godown): Ye sirf ek simple text file hai jahan passwords aur URL safe rakhe jate hain (jaise application.properties).

** setting.py (The Delivery Boy/Security Guard): Python directly .env file ko nahi       samajhta. setting.py ka kaam hai .env file se data (URL) uthana, check karna ki wo sahi hai ya nahi, aur usko Python Variables mein convert karna.

** db.py (The Connection Engine): Ab setting.py ke paas URL aa gaya hai. db.py us URL ka use karke actual database (PostgreSQL) ke sath connection establish karta hai.


## pip install SQLAlchemy:---> es command ke help se ham SQLAlchemy install krte hai bhai jiske help se Hm apne database ke sath interact krte hai aur apne data models ko define krte hai. SQLAlchemy ek powerful ORM (Object-Relational Mapping) library hai jo hamare Python code ko database ke sath seamlessly integrate krta hai, jisse ham apne database operations ko easily manage kr sakte hai.




# pip install psycopg2:---> es command ke help se ham psycopg2 install krte hai bhai jiske help se Hm apne Python application ko PostgreSQL database ke sath connect krte hai. Psycopg2 ek popular PostgreSQL adapter hai jo hamare Python code ko PostgreSQL database ke sath interact krne me madad krta hai, jisse ham apne database operations ko easily manage kr sakte hai.


# pip install pydantic_Setting ---> es command ke help se ham pydantic ke settings module ko install karenge, jisse ham apne application ke configuration settings ko manage kar sakte hai. Is module ka use karke ham apne application ke configuration settings ko easily manage kar sakte hai aur unhe environment variables ya configuration files ke through set kar sakte hai. Isse ham apne application ke configuration management ko simplify kar sakte hai aur apne application ke different environments ke liye alag-alag settings define kar sakte hai.




# fastapi dev main.py :-- command help krta hai ki hamare main.py file me koi changes hone par server automatically restart ho jayega aur hamare changes reflect ho jayenge. Isse hamara development process bahut hi smooth ho jata hai kyunki hame baar baar server ko manually restart karne ki jarurat nahi hoti hai. Is command ka use karne ke liye hame fastapi library ko install karna hoga, jiske baad ham is command ka use karke apne application ko run kar sakte hai.

# pip install PyJWT es line se ham PyJWT library ko install karenge, jisse ham apne application me JSON Web Tokens (JWT) ka use kar sakte hai. JWT ek open standard hai jo secure way me information exchange karne ke liye use hota hai. Is library ko install karke ham apne application me JWT authentication implement kar sakte hai, jisse ham apne endpoints ko secure kar sakte hai aur authorized users ko hi access de sakte hai.  



# pip install "pwdlib[argon2]"  ye line se ham pwdlib library ko install karenge, jisme argon2 hashing algorithm included hai. Argon2 ek secure password hashing algorithm hai jo hamare users ke passwords ko securely store karne me madad karega. Is library ko install karne ke baad, ham apne application me argon2 hashing algorithm ka use karke users ke passwords ko hash kar sakte hai, jisse hamare application ki security badh jayegi.
